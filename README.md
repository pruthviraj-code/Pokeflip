# 🎮 PokéFlip – Pokémon Memory Card Game

PokéFlip is a fan-made Pokémon memory card game built to test your memory and concentration. Flip cards, match Pokémon pairs, and complete the board in the fewest moves possible.

## ✨ Features
- 🃏 Memory card matching gameplay
- ⚛️ Built with **React** and **JavaScript**
- 🎨 Styled using **Tailwind CSS**
- 🌙 Light & Dark mode
- 🌐 Pokémon data fetched using **PokéAPI**
- 💾 Progress and settings saved with **Local Storage**
- 📱 Responsive design
- 🎯 Multiple difficulty levels
- ⏱️ Move counter and game timer
- 🔄 Randomized card shuffle every game

## 🛠️ Tech Stack
- React
- JavaScript (ES6+)
- Tailwind CSS
- PokéAPI
- Local Storage
